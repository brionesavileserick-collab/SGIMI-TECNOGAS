# Guía rápida de uso de SGIMI TECNOGAS

## 1. ¿Cómo abrir la app?
- En Windows, haz doble clic en el archivo: SGIMI_TECNOGAS.exe
- En macOS o Linux, abre el archivo ejecutable correspondiente
- Si aparece una advertencia de seguridad, confirma que deseas abrirlo

## 2. ¿Qué verás al abrirla por primera vez?
La aplicación te pedirá crear el primer usuario del sistema.
Complete estos datos:
- Nombre
- Correo
- Contraseña
- Confirmar contraseña

Luego presiona Crear usuario.

## 3. ¿Cómo entrar después?
- Escribe tu correo
- Escribe tu contraseña
- Presiona Iniciar Sesion

## 4. ¿Qué puedes hacer dentro?
- Ver dashboard con estadísticas
- Gestionar productos
- Gestionar sucursales
- Registrar inventario
- Ver movimientos
- Revisar alertas
- Consultar historial y reportes

## 5. ¿Dónde se guardan los datos?
Los datos se guardan automáticamente en la carpeta de datos del usuario del sistema, por lo que no necesitas instalar nada extra.

## 6. Si algo no funciona
- Asegúrate de tener permisos para ejecutar el archivo
- Si la app no abre, vuelve a descargar el ejecutable
- Si persiste el problema, contacta al soporte técnico
